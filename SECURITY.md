# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.2.x   | Yes       |
| < 0.2   | No        |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Email **tushar@onllm.dev** with:

- Description of the vulnerability
- Steps to reproduce
- Potential impact

### What to Expect

- **Acknowledgment** within 48 hours
- **Status update** within 7 days
- **Fix or mitigation** as soon as practical

## Scope

This policy covers the `tribe` Python package. Upstream dependencies (PyTorch, Transformers, eugenehp/tribev2, tribev2-rs) should be reported to their respective maintainers.
